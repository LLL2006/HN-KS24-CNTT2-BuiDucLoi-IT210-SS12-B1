Nó được gọi là Starter vì nó đóng vai trò là điểm khởi đầu nhanh chóng. Thay vì phải đi tìm từng phiên bản thư viện tương thích với nhau, bạn chỉ cần khai báo một Starter duy nhất

Những thư viện nhỏ lẻ mà nó đã gom nhóm:

Spring MVC: Các thư viện cốt lõi để xây dựng ứng dụng web 

Embedded Tomcat: Máy chủ được nhúng trực tiếp vào file JAR, giúp bạn chạy app bằng lệnh main mà không cần cài Tomcat rời.

Jackson: Thư viện tự động chuyển đổi dữ liệu Java Object sang JSON và ngược lại.

Spring Boot Starter Logging: Hệ thống ghi log

Spring Validation: Hỗ trợ kiểm tra dữ liệu đầu vào.
